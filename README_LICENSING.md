# Conductor Web Licensing Pack

This repository has been prepared for a closed-source commercial product. The recommended position is:

- Conductor Web itself is proprietary software owned by Paul Walker / Mav3r1ckmediastudio.
- Customers should receive use rights through a separate written commercial licence or EULA.
- Third-party libraries and map services remain under their own licences and terms.
- Map attribution must remain visible anywhere maps are displayed.
- Real API keys must not be committed to the repository.

## Files included

| File | Purpose |
|---|---|
| `LICENSE` | Proprietary all-rights-reserved source-code licence for the repo. |
| `NOTICE` | Short ownership and notice file. |
| `COMMERCIAL_EULA_TEMPLATE.md` | Customer-facing commercial licence / EULA template. |
| `THIRD_PARTY_NOTICES.md` | Open-source and third-party notices. |
| `ATTRIBUTION.md` | Required map/provider attribution guidance. |
| `SECURITY_AND_KEYS.md` | Practical notes for API keys and deployment secrets. |
| `RELEASE_CHECKLIST.md` | Pre-release checklist before putting the repo or product in front of customers. |
| `frontend/.env.example` | Safe placeholder for environment variables. |

## Important

These documents are practical commercial templates, not legal advice. Before selling Conductor Web under a signed customer agreement, have a solicitor review the EULA, warranty, liability, data-protection, and support terms.
